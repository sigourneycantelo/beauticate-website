# Adding products

> How to add or feature a product so it renders as a Beauticate card. For Claude and Claude Code. Read CLAUDE.md section 8 for the card rules. This file is the steps.

## When to use this

When someone adds a product to the shop, or features a product or affiliate inside a story or an edit.

## What I need from you

- Product name and brand.
- Price.
- Whether it's in our Shopify shop, or an affiliate.
- The link: the Shopify product handle for our own, or the retailer URL and retailer name for an affiliate.
- The image, and which type it is: deep-etched product, a model wearing or using it, or an Instagram shot.
- For an Instagram shot, the creator's handle so we can credit them.

If any are missing, ask. Never invent prices, links or retailer names.

## Steps

1. Build the card from CLAUDE.md section 8. No frame, no button. The whole card links out, and a save heart sits top-right.
2. Own product: pull data and image from Shopify by handle. Price reads "$X". The card links to our Shopify product page.
3. Affiliate: price reads "$X at [retailer]" with the small arrow. The card opens the retailer in a new tab.
4. Set the image to its type. Deep-etched sits on the beige tile `#EEE9E1`. A model or Instagram shot fills the tile edge to edge. An Instagram shot credits the creator with their handle.
5. In any edit or row that mixes our products and affiliates, add the footnote "Some links are affiliate links."
6. Add plain, descriptive alt text.

## Rules that always apply

- Card rules: CLAUDE.md section 8.
- Own products link to our Shopify page. Affiliates show "at [retailer]" and open in a new tab.
- Never invent a price, a link or a retailer. Ask.
- Credit the creator on any Instagram-sourced image.

## Definition of done

- [ ] Card built to section 8, no frame, no button, save heart
- [ ] Own vs affiliate handled correctly (destination and price line)
- [ ] Image type set, deep-etched on beige or full-bleed photo
- [ ] Creator credited on any Instagram shot
- [ ] Affiliate-links footnote added if the edit is mixed
- [ ] Alt text added

## If something's missing

Ask in one short list. Usual gaps: the price, the link, the retailer name for affiliates, the image, and the creator handle for Instagram shots.

## Example request

> Add three products to the Selfcare Sunday edit. The buj Uplifter and the Tulita Agati are in our shop. The Theragun Mini is an Amazon affiliate at $299. Use the deep-etched shots for ours and the Instagram shot for the Theragun, credit @therabody.
